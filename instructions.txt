# Telegram Channel Reporter Package

## Requirements
- Python 3.8+
- Telethon library (`pip install telethon`)
- GCC compiler (for compiling C program)

## Setup
1. Edit your scripts:
   - `report_channel.py`, `report_channel_fast.py`, and `multi_account_reporter.py`
   - Insert your `api_id`, `api_hash`, and session names where needed

2. (First time only)
   Run scripts to create login sessions per account (a small `.session` file will be created).

3. Prepare `channels.txt` like:
   https://t.me/example1
   https://t.me/example2

## How to Use
### Compile the C launcher
```bash
gcc report_channel.c -o report_channel
```

### Run the C launcher:
```bash
./report_channel
```

Or use Python scripts directly.
